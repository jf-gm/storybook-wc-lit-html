# Storybook with Web Components: Lit-HTML

## Setup the project

Create a directory for your project
- `mkdir my-project`
- `cd my-project`

Initialize a npm project
- `npm init --yes`

Add dev server dependency for storybook
- `yarn add -D @web/dev-server-storybook`

Create the storybook project
- `npx sb init --type web_components`

Run the project
- `yarn storybook`
